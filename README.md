# BeachInfo
## Introduction
해수욕장 혼잡도를 신호등 형식으로 볼 수 있는 앱입니다.

## Composition
- [해수욕장 혼잡도 API](https://www.data.go.kr/data/15061784/openapi.do)를 사용하여 혼잡도를 신호등으로 구현했습니다.
- 해수욕장 이름 클릭 시 해당 장소를 Google Map에서 보여주도록 구현했습니다.
